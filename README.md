<h1>Easy Shopping</h1>
<br>
<h2>Esse é um projeto que aprendi no <a href="https://rodolfomori.com.br/">DevClub</a></h2>
<br>

Este projeto foi criado com o objetivo de fortalecer fundamentos essenciais do desenvolvimento web,
priorizando:

· Estrutura semântica
<br>
· Responsividade
<br>
· Flexbox para construção de layout
<br>
· Organização de classes
<br>
· Código limpo e legível

Mais do que apenas reproduzir uma interface, a proposta foi desenvolver um layout pensando como um
produto real - garantindo melhor experiência para o usuário e facilidade de manutenção do código.
